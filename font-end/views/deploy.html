# IE浏览器开发注意事项

注意

禁止使用es5和es6的代码，IE中踩过的坑：

  1. object.assign()可能会导致ie浏览器无法识别导致报错；
  2. 不能使用let const声明变量；
  3. 禁止使用箭头函数；
  4. for...in遍历数据时，在ie中使用可能会导致出错
